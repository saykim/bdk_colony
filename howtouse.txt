애플리케이션이 시작되면, Gradio가 제공하는 로컬 호스트 주소(예: http://127.0.0.1:7860)로 웹 브라우저에서 접속하여 인터페이스를 사용할 수 있습니다.

사용 단계
이미지 업로드: "Upload Image" 섹션에서 분석할 페트리 접시 이미지를 선택하고 업로드합니다.
탐지 방법 선택: 현재는 FastSAM만 사용할 수 있습니다.
이미지 분석: "🔍 Analyze Image" 버튼을 클릭하여 감지 과정을 시작합니다.
수동 수정:
콜로니 추가: 이미지를 클릭하여 누락된 콜로니를 추가합니다. 수동으로 추가된 콜로니의 번호는 빨간색으로 표시됩니다.
콜로니 제거: "🔄 Toggle Edit Mode" 버튼을 클릭하여 제거 모드로 전환한 후, 기존 콜로니를 클릭하여 제거합니다.
최근 추가 포인트 되돌리기: "↩️ Undo Last Point" 버튼을 클릭하여 가장 최근에 추가한 콜로니 포인트를 제거합니다.
분석 설정 조정: "⚙️ Analysis Settings" 아코디언을 확장하여 입력 크기, IOU 임계값, 신뢰도 임계값, 크기 백분위수, 원형도 임계값 등의 파라미터를 조정하여 감지 정확도를 최적화합니다.
